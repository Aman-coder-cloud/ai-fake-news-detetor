📘 About This Project

An AI-powered Fake News Detection System built using Machine Learning, Natural Language Processing, Flask, and Python.

🎯 Project Objective

The objective of this project is to identify whether a news article is Real or Fake using Machine Learning algorithms and NLP techniques.


🛠 Technologies Used

  • Python
  • Flask
  • Scikit-Learn
  • TF-IDF Vectorizer
  • Logistic Regression
  • SQLite Database
  • Matplotlib
  • HTML & CSS

📊 Model Information

Dataset Size: 44,000+ Articles

Accuracy: 98.58%

Prediction Method: TF-IDF + Logistic Regression
